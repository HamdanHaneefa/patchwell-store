### Run Locally
```bash

```

### Build for Production
```bash

npm start
```

### Deploy to Vercel
```bash
npm i -g vercel
vercel
```

## 🔧 Configuration

### Works Out of the Box
The application runs with mock data immediately - no configuration required!

### Optional: Connect Shopify
1. Get Shopify Storefront API credentials
2. Add to `.env.local`:
   ```env
   NEXT_PUBLIC_SHOPIFY_STORE_DOMAIN=your-store.myshopify.com
   NEXT_PUBLIC_SHOPIFY_STOREFRONT_ACCESS_TOKEN=your_token
   ```
3. Restart server

## 🎨 Customization

### Easy Changes
- **Logo**: Replace `public/images/logo.png`
- **Colors**: Edit CSS variables in `app/globals.css`
- **Products**: Modify `MOCK_PRODUCTS` in `lib/shopify/index.ts`
- **Content**: Edit page components in `app/`
